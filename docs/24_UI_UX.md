## Design-Principes

1. **Rust door reductie** — Één primaire actie per scherm
2. **Witruimte** — Ruim, niet dicht
3. **Snelheid** — Optimistic UI, < 100ms feedback
4. **Command-first** — Cmd+K palette
5. **Motion met betekenis** — Subtiele animaties (150–250ms)
6. **Premium voel** — Animatie, micro-interactions
7. **Accessibility** — WCAG 2.1 AA
